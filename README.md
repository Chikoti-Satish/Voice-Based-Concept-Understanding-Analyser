# 🎙️ Voice-Based Concept Understanding Analyser (VBCUA)

An AI-powered web application that evaluates a student's conceptual understanding by analyzing spoken explanations. The system converts speech to text, compares the explanation with reference concepts using semantic similarity, analyzes voice features (like filler words and speech energy), and generates an automated, AI-driven qualitative evaluation report.

---

## 📌 Project Overview

Traditional assessments mainly evaluate written responses and often fail to measure a student's conceptual understanding through verbal communication.

The **Voice-Based Concept Understanding Analyser** addresses this challenge by allowing students to explain concepts through speech. The application automatically transcribes the audio, evaluates semantic similarity with reference concepts, analyzes audio characteristics, generates customized AI feedback, and produces a downloadable PDF report.

---

## ✨ Features

- 🎤 Voice-based concept evaluation
- 📝 Automatic Speech-to-Text transcription (OpenAI Whisper)
- 🧠 Semantic similarity analysis using Sentence Transformers (SBERT)
- 📊 AI-based qualitative feedback generation (Google Gemini API)
- 📈 Audio waveform and feature visualization
- 📄 Automatic structured PDF report generation
- 🌐 Interactive Streamlit web interface
- ⚡ Fast and lightweight deployment

🚀 Installation & Setup
1. Clone the repository
Bash
git clone [https://github.com/Chikoti-Satish/Voice-Based-Concept-Understanding-Analyser.git](https://github.com/Chikoti-Satish/Voice-Based-Concept-Understanding-Analyser.git)
cd Voice-Based-Concept-Understanding-Analyser
2. Install FFmpeg (Required for Audio Processing)
Windows: Open terminal and run winget install Gyan.FFmpeg

Mac: Run brew install ffmpeg

Linux: Run sudo apt update && sudo apt install ffmpeg

3. Set up a Virtual Environment
Bash
# Windows
python -m venv env
.\env\Scripts\activate

# Mac/Linux
python3 -m venv env
source env/bin/activate
4. Install Dependencies
Navigate into the Source folder and install requirements:

Bash
cd Source
pip install -r requirements.txt
5. Configure API Keys (Crucial Step)
This project requires a Google Gemini API key to generate qualitative feedback.

Navigate to Source/.streamlit/

Rename secrets.toml.example to secrets.toml

Add your Gemini API key:

Ini, TOML
GEMINI_API_KEY = "your_actual_api_key_here"
6. Run the Application
Make sure you are in the Source folder, then run:

Bash
streamlit run app.py
🎯 Workflow
Open the Streamlit application.

Enter the reference concept/topic text.

Upload the student's recorded audio explanation.

The system converts the speech into text.

The system calculates semantic similarity (0-100%).

Audio features (speech energy, waveform) are plotted.

The Gemini API analyzes the metrics and generates tailored feedback.

View the final evaluation and download the comprehensive PDF report.
